# WORKS・GALLERY 更新マニュアル

> VPS（レンタルサーバー）での運用を前提としています。

---

## 基本の流れ

```
1. 管理画面で記事・作品を登録・編集
       ↓
2. 「保存」ボタンをクリック
       ↓
3. サーバーに自動反映（反映完了！）
```

---

## WORKS を更新する

### 記事を追加する

1. ブラウザで `https://ドメイン名/admin/index.html` を開く
2. 「WORKS管理」タブ → 「**+ 新規作成**」
3. 各フィールドを入力する

| フィールド | 内容 |
|---|---|
| ID | 記事ID（例: `007`）。他と重複しない連番 |
| 日付 | 表示用（例: `2026.06.01`） |
| カテゴリ | PROJECT / CONTEST / OTHER |
| タイトル | 記事タイトル |
| スケール | 例: `1/8 SCALE · RESIN KIT` |
| 抜粋 | 一覧カードに表示される短い説明文 |
| サムネイル画像パス | 例: `images/works/thumb07.webp` |
| メインビジュアル画像パス | 詳細ページの大きい画像 |
| NEWバッジ | チェックするとNEWフィルタに表示 |

4. **本文コンテンツ**をブロック単位で追加
   - `+ テキスト段落` → 本文テキスト
   - `+ 見出しH2` → セクション見出し
   - `+ 画像×2 / ×3` → インライン画像グリッド
5. 「**保存**」をクリック

### 記事を編集・削除する

- 「**編集**」→ 内容が読み込まれる → 変更後「保存」
- 「**削除**」→ 確認後削除

## GALLERY を更新する

### 作品を追加する

1. 「GALLERY管理」タブ → 「**+ 新規作成**」
2. 各フィールドを入力する

| フィールド | 内容 |
|---|---|
| ID | 作品ID（例: `g006`）。他と重複しない |
| タイトル | 作品名 |
| スケール | `1/8` のように分母だけ入力 |
| 年 | 例: `2026` |
| メイン画像パス | 縦長メイン画像のパス |
| サブ画像1〜4 | 右側2×2グリッドのサブ画像 |
| キャプション | 使用塗料・制作コメントなど |

3. 「**保存**」をクリック

---

## 最速の更新手順

1. 管理画面で記事・作品を登録・編集
2. 「**保存**」ボタンをクリック
3. 反映完了！（自動でサーバーのデータが更新されます）

---

## 画像のアップロード（自動入力）

各入力欄の横にある「**UP**」ボタンを使うのが最も簡単です。

1. 入力欄横の「**UP**」をタップ
2. 画像を選択
3. **自動でパスが入力**され、アップロード完了です。

> [!TIP]
> 1枚あたり **最大 30MB** まで対応。撮影した写真をそのままアップロードできます。  
> 複数の画像をまとめて管理・命名したい場合は「画像アップロード」タブも併用可能です。
