# WORKS・GALLERY 更新マニュアル

> VPS（レンタルサーバー）での運用を前提としています。

---

## 基本の流れ

```
① 管理画面で記事・作品を登録・編集
       ↓
② JSONをダウンロード
       ↓
③ FTPでVPSにアップロード（JSONと画像）
       ↓
④ ブラウザでサイトを確認
```

---

## WORKS を更新する

### 記事を追加する

1. ブラウザで `https://ドメイン名/admin/index.html` を開く
2. 「WORKS管理」タブ → 「**+ 新規作成**」
3. 各フィールドを入力する

| フィールド | 内容 |
|---|---|
| ID | 記事ID（例: `007`）。他と重複しない連番 |
| 日付 | 表示用（例: `2026.06.01`） |
| カテゴリ | PROJECT / CONTEST / OTHER |
| タイトル | 記事タイトル |
| スケール | 例: `1/8 SCALE · RESIN KIT` |
| 抜粋 | 一覧カードに表示される短い説明文 |
| サムネイル画像パス | 例: `images/works/thumb07.webp` |
| メインビジュアル画像パス | 詳細ページの大きい画像 |
| NEWバッジ | チェックするとNEWフィルタに表示 |

4. **本文コンテンツ**をブロック単位で追加
   - `+ テキスト段落` → 本文テキスト
   - `+ 見出しH2` → セクション見出し
   - `+ 画像×2 / ×3` → インライン画像グリッド
5. 「**保存**」をクリック

### 記事を編集・削除する

- 「**編集**」→ 内容が読み込まれる → 変更後「保存」
- 「**削除**」→ 確認後削除

## GALLERY を更新する

### 作品を追加する

1. 「GALLERY管理」タブ → 「**+ 新規作成**」
2. 各フィールドを入力する

| フィールド | 内容 |
|---|---|
| ID | 作品ID（例: `g006`）。他と重複しない |
| タイトル | 作品名 |
| スケール | `1/8` のように分母だけ入力 |
| 年 | 例: `2026` |
| メイン画像パス | 縦長メイン画像のパス |
| サブ画像1〜4 | 右側2×2グリッドのサブ画像 |
| キャプション | 使用塗料・制作コメントなど |

3. 「**保存**」をクリック

---

## サイトに反映する（JSONアップロード）

管理画面での保存はブラウザ内の一時保存です。  
**JSONをダウンロードしてVPSにアップロードして初めて反映されます。**

### 手順

1. 「**JSONエクスポート**」タブを開く
2. 更新内容に応じてクリック
   - `works.json を生成` → WORKS更新時
   - `gallery.json を生成` → GALLERY更新時
3. 「**ファイルにダウンロード**」
4. FTPクライアント（FileZilla等）でVPSの **`data/`** フォルダに**上書きアップロード**
5. ブラウザでサイトをリロードして確認

---

## 画像をアップロードする

| 画像の種類 | アップロード先パス |
|---|---|
| WORKSサムネイル | `images/works/thumb01.webp` |
| WORKS詳細メイン | `images/works/detail01_main.webp` |
| WORKS詳細インライン | `images/works/detail01_a.webp` |
| GALLERYメイン | `images/works/01_main.webp` |
| GALLERYサブ | `images/works/01_sub1.webp` |

> [!TIP]
> FTPで `images/works/` にアップロードしてから、管理画面でパスを入力する。  
> 推奨形式は **WebP**（軽量）。ファイル名は既存の連番ルールに合わせる。

## ファイル構成（参考）

```
kaien2/（サーバールート）
├── admin/
│   ├── index.html        ← 管理画面
│   └── 更新マニュアル.md
├── data/
│   ├── works.json        ← WORKS記事データ  ★更新対象
│   └── gallery.json      ← GALLERY作品データ ★更新対象
├── images/
│   └── works/            ← 画像置き場       ★更新対象
├── index.html
├── works.html
├── gallery.html
└── worksdetail.html
```
